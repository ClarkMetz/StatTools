Useful Excel Formulas

Descriptive Statistics:

  • AVERAGE(number1, [number2], ...): Calculates the arithmetic mean of a range of numbers. Example: =AVERAGE(A1:A10) calculates the average of the values in cells A1 through A10.
  • MEDIAN(number1, [number2], ...): Returns the median (middle value) of a range of numbers. Example: =MEDIAN(B1:B5)
  • MODE.SNGL(number1,[number2],...): Returns the most frequently occurring number in a data set. For multiple modes, use MODE.MULT(). Example: =MODE.SNGL(C1:C8)
  • STDEV.S(number1,[number2],...): Calculates the sample standard deviation. Use STDEV.P() for population standard deviation. Example: =STDEV.S(D1:D20)
  • VAR.S(number1,[number2],...): Calculates the sample variance. Use VAR.P() for population variance. Example: =VAR.S(E1:E15)
  • QUARTILE.INC(array,quart): Returns the quartile of a dataset. quart is 1, 2, or 3 for the 25th, 50th (median), or 75th percentile, respectively. Example: =QUARTILE.INC(A1:A10,1) for the first quartile.
  • PERCENTILE.INC(array,k): Returns the k-th percentile of values in a range, where k is between 0 and 1 inclusive. Example: =PERCENTILE.INC(A1:A10,0.5) for the 50th percentile (median).
  • COUNT(value1, [value2], ...): Counts the number of cells that contain numbers. Example: =COUNT(A1:A10)
  • COUNTIF(range, criteria): Counts the number of cells within a range that meet a given criterion. Example: =COUNTIF(A1:A10,">10") counts cells greater than 10.
  • FREQUENCY(data_array, bins_array): Calculates how often values occur within a range of values (bins) and returns a vertical array of numbers. This is useful for creating histograms. Requires selecting a range of cells for the output and entering the formula as an array formula (Ctrl+Shift+Enter).

Probability:

  • RAND(): Generates a random number between 0 and 1.
  • RANDBETWEEN(bottom, top): Generates a random integer between the specified bottom and top values.
  • COMBIN(number, number_chosen): Calculates the number of combinations (without regard to order). Example: =COMBIN(5,2) calculates "5 choose 2."
  • PERMUT(number, number_chosen): Calculates the number of permutations (with regard to order). Example: =PERMUT(5,2)
  • BINOM.DIST(number_s, trials, probability_s, cumulative): Returns the binomial probability distribution. If cumulative is TRUE, it returns the cumulative distribution function; if FALSE, it returns the probability mass function.

Normal Distribution:

  • NORM.DIST(x, mean, standard_dev, cumulative): Returns the normal distribution for the specified mean and standard deviation. If cumulative is TRUE, it returns the cumulative distribution function; if FALSE, it returns the probability density function.
  • NORM.S.DIST(z, cumulative): Returns the standard normal distribution.
  • NORM.INV(probability, mean, standard_dev): Returns the inverse of the normal cumulative distribution (useful for finding critical values).
  • NORM.S.INV(probability): Returns the inverse of the standard normal cumulative distribution.

Inference (Confidence Intervals and Hypothesis Testing):

  • T.DIST(x,deg_freedom,cumulative): Returns the Student's t-distribution.
  • T.INV(probability,deg_freedom): Returns the inverse of the Student's t-distribution (for two-tailed tests, use 1-probability/2).
  • T.TEST(array1,array2,tails,type): Performs a t-test. tails is 1 or 2 for one-tailed or two-tailed tests. type specifies the type of t-test (1=paired, 2=two-sample equal variance, 3=two-sample unequal variance).
  • CHISQ.TEST(actual_range,expected_range): Performs a chi-square test of independence.
  • CHISQ.DIST(x,deg_freedom,cumulative): Returns the chi-squared distribution.
  • CHISQ.INV(probability,deg_freedom): Returns the inverse of the chi-squared distribution.

Correlation and Regression:

  • CORREL(array1, array2): Calculates the Pearson correlation coefficient.
  • SLOPE(known_y's, known_x's): Calculates the slope of the linear regression line.
  • INTERCEPT(known_y's, known_x's): Calculates the y-intercept of the linear regression line.
  • LINEST(known_y's, known_x's, [const], [stats]): Returns an array of regression statistics, including the slope, intercept, R-squared, and more. If const is TRUE or omitted, calculates the intercept; if FALSE, forces the intercept to be zero. If stats is TRUE, returns additional regression statistics. This is a powerful function but requires selecting a range of cells for the output and entering it as an array formula (Ctrl+Shift+Enter).
  • FORECAST.LINEAR(x, known_y's, known_x's): Calculates a predicted y-value for a given x-value based on the linear regression line. (In newer versions of Excel, this is FORECAST()).
  • RSQ(known_y's, known_x's): Calculates the R-squared value for a linear regression.
